GUIA RAPIDO DE LIGACAO - TANG NANO 9K
=====================================

Este guia reflete a montagem que funcionou no protoboard.


PINOS DA TANG USADOS
--------------------
- 25
- 26
- 27
- 28
- 29
- 30
- 41
- 42
- 3V3
- GND


ALIMENTACAO
-----------
- Tang 3V3 -> trilha positiva
- Tang GND -> trilha negativa


BOTOES DO CODIGO
----------------
- Tang 25 -> proto a1
- Tang 26 -> proto a6
- Tang 27 -> proto a11
- Tang 28 -> proto a16

- trilha positiva -> proto j3
- trilha positiva -> proto j8
- trilha positiva -> proto j13
- trilha positiva -> proto j18

Observacao:
- os 4 botoes do codigo usam apenas sinal de um lado e 3V3 do outro
- nao precisa resistor nos botoes
- o GND dos botoes nao e usado, porque os pinos do FPGA estao com pull-down interno


LEDS DE VISUALIZACAO DOS BITS
-----------------------------
LED 1
- Tang 29 -> proto a23
- resistor 1 -> b23, i23
- LED 1 -> perna maior j23, perna menor j24
- trilha negativa -> proto i24

LED 2
- Tang 30 -> proto a25
- resistor 2 -> b25, i25
- LED 2 -> perna maior j25, perna menor j26
- trilha negativa -> proto i26

LED 3
- Tang 41 -> proto a27
- resistor 3 -> b27, i27
- LED 3 -> perna maior j27, perna menor j28
- trilha negativa -> proto i28

LED 4
- Tang 42 -> proto a29
- resistor 4 -> b29, i29
- LED 4 -> perna maior j29, perna menor j30
- trilha negativa -> proto i30

TESTE DO CODIGO CORRETO
-----------------------
Com os botoes em modo toggle:

1. aperte S2 para limpar tudo
2. toque no botao do bit 3
3. toque no botao do bit 1
4. toque no botao do bit 0
5. nao toque no botao do bit 2
6. aperte S1 para confirmar

Codigo correto: 1011


CONTADOR DE TENTATIVAS
----------------------
O contador usa representacao binaria de 3 bits, conforme o enunciado.

Leitura:

- bit 0 = primeiro LED do contador
- bit 1 = segundo LED do contador
- bit 2 = terceiro LED do contador

Exemplos:

- 1 tentativa = 001
- 2 tentativas = 010
- 3 tentativas = 011
- 4 tentativas = 100
- 5 tentativas = 101
