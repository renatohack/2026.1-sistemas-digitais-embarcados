GUIA RAPIDO DE LIGACAO - TANG NANO 9K
=====================================

USE ESTE GUIA PARA BOTAO DE 4 PERNAS
------------------------------------
Em cada botao, use pontos diagonais:

- lado do sinal: ponto da esquerda
- lado do 3V3: ponto da direita na outra linha

TANG NANO 9K - PINOS USADOS
---------------------------
Use exatamente estes pinos escritos na placa:

- 3V3
- 25
- 26
- 27
- 28


BOTOES NO PROTOBOARD
--------------------
Considerando o padrao que voce descreveu:

- botao do bit 0: d1, d4, g1, g4
- botao do bit 1: d7, d9, g7, g9
- botao do bit 2: d13, d15, g13, g15
- botao do bit 3: d19, d21, g19, g21


JUMPERS
-------
Ligue assim:

1. Tang 25   -> a1
2. Tang 26   -> a7
3. Tang 27   -> a13
4. Tang 28   -> a19

5. Tang 3V3  -> j4
6. i4        -> j9
7. i9        -> j15
8. i15       -> j21


JUMPERS DO 3V3
--------------
Para o 3V3, use exatamente assim:

1. um lado no pino 3V3 da Tang, outro lado em j4
2. um lado em i4, outro lado em j9
3. um lado em i9, outro lado em j15
4. um lado em i15, outro lado em j21

No fim, o 3V3 fica presente em:

- j4
- j9
- j15
- j21


RESUMO DIRETO
-------------
- pino 25 da Tang = botao do bit 0
- pino 26 da Tang = botao do bit 1
- pino 27 da Tang = botao do bit 2
- pino 28 da Tang = botao do bit 3

- bit 0 usa o botao em d1/d4/g1/g4
- bit 1 usa o botao em d7/d9/g7/g9
- bit 2 usa o botao em d13/d15/g13/g15
- bit 3 usa o botao em d19/d21/g19/g21

- o lado esquerdo dos botoes recebe os sinais em:
  a1, a7, a13, a19

- o lado direito dos botoes recebe 3V3 em:
  j4, j9, j15, j21


CODIGO 1011
-----------
Para testar o codigo correto:

- aperte uma vez o botao do bit 3
- aperte uma vez o botao do bit 1
- aperte uma vez o botao do bit 0
- nao aperte o botao do bit 2
- depois aperte o botao onboard de confirmacao

Agora os botoes do protoboard funcionam como TOGGLE:

- um toque muda de 0 para 1
- outro toque no mesmo botao muda de 1 para 0
- o estado fica guardado ate voce mudar de novo ou apertar reset

Fluxo pratico:

1. aperte S2 para limpar tudo
2. toque no botao do bit 3
3. toque no botao do bit 1
4. toque no botao do bit 0
5. aperte S1 para confirmar


OBSERVACAO
----------
Este guia assume que os 4 botoes seguem o mesmo formato e a mesma distribuicao
que voce descreveu para os dois primeiros.

O motivo de usar a1, a7, a13 e a19 e j4, j9, j15 e j21 e que esses pontos
correspondem a cantos diagonais livres dos botoes de 4 pernas.
